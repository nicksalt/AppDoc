AppDoc allows you to test your hearing, visual acuity, and colour vision. There are several components of the application, select the one you need help with below:

  • Home Screen
  • Navagation Bar
  • Hearing Test
  • Colour Test
  • Eye Test
  • Results
  • Settings
  • Account
  • Notes

 

Home Screen

It was our goal to make the Home Page as user-friendly as possible. Simply select one of the test you would like to complete and it will initiate.

Navagation Bar

Layout a walkway, with visible start and end point, and measure the walkway length. Have the person stand at the start point, facing along the walkway.

Navagation Bar

  1. Selecting "Home" returns you to the Home Screen
  2. Selecting "Results" take you to the Results page
  3. Selecting "Help" brought you here!
  4. Selecting "Sign Out" signs you out of you Account, for more information see Account
